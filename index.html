About Me

Halo semua kenalin nama saya Kevin Ardiansyah, Lahir di Batam, 19 September 2004, saya anak kedua dari tiga bersaudara. Hobi saya adalah bermain game. Saya biasa di panggil Kepin atau juga Ouchy, dikesempatan kali ini saya mau menceritakan pengalaman saya, yaitu hobi saya yang bisa menghasilkan uang. Ya saya sudah jelaskan di awal tadi hobi saya adalah bermain game. Dan dengan bermain game online saya bisa menghasilkan uang. Awal saya bermain game online (2017) karena saya merasa bosan bermain di luar rumah, dan pada waktu itu saya di belikan handphone oleh orang tua saya, ketika saya di belikan handphone, saya langsung menggunakan handphone itu untuk bermain game online karena saya sangat suka bermain game.

Game online yang pertama saya mainkan di handphone saya adalah game FREE FIRE biasa di sebut game 8BIT game FREE FIRE ini adalah game dengan genre survival atau battle royale. Setahun setelah saya bermain game FREE FIRE saya mulai mengikuti kompetitif yang ada di dalam game tersebut. Pada awal saya mengikuti kompetitif di game tersebut, saya mempunyai team yang bernama TEAM QL yang anggota di isi oleh saya sendiri dan 4 rekan saya yang bernama Fyroz(C), Arrafi, Ari, dan Tijey dengan keempat rekan team saya ini, saya berjuang bersama mereka untuk memenangkan turnamen - turnamen yang ada di dalam game tersebut. Dua tahun saya berkompetitif bisa di bilang belum mendapatkan hasil yang memuaskan.
